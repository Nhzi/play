Play qualifies because it serves as an accessible on-ramp to the Arc ecosystem, combining onchain identity, gaming, and user engagement into a single platform. The project allows users to create persistent onchain profiles, participate in games, and mint achievements directly on Arc Testnet, helping introduce blockchain interactions in a familiar and engaging way.

Beyond gaming, Play is part of a broader ecosystem that includes upcoming consumer applications such as Tanget Wallet. Play helps acquire and retain users through entertainment while establishing wallet-linked identities that can be used across future products. This creates a foundation for deeper blockchain adoption and user retention within the ecosystem.

The project is also technically significant because it demonstrates practical use of smart contracts, wallet authentication, onchain achievements, and user-owned profiles while maintaining a user-friendly experience. Future development includes AI-powered gaming experiences, intelligent assistants, and natural-language blockchain interactions through Tanget Wallet, making blockchain technology more accessible to non-technical users.
As the ecosystem grows, Play will serve as both an engagement platform and a gateway into more advanced blockchain applications, helping expand adoption on Arc and showcasing real-world consumer use cases.

API credits will be used to power AI-driven features across the Tanget Wallet and Play ecosystem, primarily focused on natural language understanding and transaction assistance.
In Tanget Wallet, AI will be used to convert natural language inputs into structured blockchain actions. This includes parsing user intent (e.g. transfers, swaps, bridging), identifying entities such as tokens and recipients, resolving ambiguity in user commands, and generating transaction parameters for user confirmation before execution. The goal is to make blockchain interactions accessible through simple conversational input rather than technical wallet operations.

AI credits will also support conversational wallet assistance, allowing users to query balances, transaction history, and portfolio insights in natural language. Additional usage includes safety checks such as anomaly detection, transaction explanation, and confirmation prompts for high-risk actions.

Within the Play ecosystem, AI will be used for user engagement features such as intelligent game assistance, personalized recommendations based on player behavior, and potential AI-powered opponents or moderation tools as the platform scales.

Overall, API usage is centered on improving usability, reducing friction in blockchain interactions, and enabling non-technical users to interact with decentralized applications through natural language interfaces.
